# [React Cours] React Router(v6) 실습

### 목차
1. 리액트 프로젝트에 라우터 적용하기
2. 페이지 컴포넌트 만들어주기
3. Route 컴포넌트로 특정 주소에 컴포넌트 연결하기
4. Navbar 만들고 Link 컴포넌트로 다른 페이지 이동하기
5. Navbar를 페이지들의 공통 레이아웃 컴포넌트로 만들기
6. URL 파라미터 사용하기
7. 쿼리 스트링 사용하기
8. 리액트 라우터 부가기능 사용하기
